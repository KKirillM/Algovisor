Algovisor. exe-модуль. Приложение аналитического терминала.

Algovisor_cli. exe-модуль. Приложение для удаления лицензии из реестра при установке терминала.

ASTS_Connector. lib-библиотека. Интерфейс для работы с протоколом ASTSBridge.

CQG_Server. exe-модуль. Приложение-сурогат для подключения к cqg.

DataCollector. exe-модуль. Приложение для сбора исторических данных с разных датафидов в edb-файлы.

edbExport. exe-модуль. Приложение для экпорта данных из edb-файлов в csv формат.

LightInstaller. exe-модуль. Приложение для скачивания и запуска установщика терминала.

Plaza2_Connector. dll-библиотека. Интерфейс для работы с протоколом Plaza2.

Plaza2_Utility. lib-библиотека. Вспомогательные интерфейсы и обёртки над Plaza2 API.

Utility. lib-библиотека. Вспомогательные интерфейсы.
